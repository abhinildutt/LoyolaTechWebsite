## Getting Started

### Prerequisites

- Node.js (v20.11.1 or higher recommended)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- **React 19** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Heroicons** - Icon library

## Sections

1. **Hero Section** - Welcome message with animated background elements
2. **Services** - Showcase of 6 main services (Blockchain, Tokens, Web3, IoT, Data Analytics, NFT Development)
3. **Values** - Company values and Custom Blockchain Development section
4. **Testimonials** - Customer testimonials
5. **Footer** - Contact information and links

## Customization

- Colors can be customized in `tailwind.config.js`
- Animations can be adjusted in component files using Framer Motion
- Content can be updated directly in component files
